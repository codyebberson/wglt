WGLT - WebGL Terminal

  • Fast, lightweight, terminal emulator using WebGL
  • 100% TypeScript
  • MIT license open source
  • ~30kb minified JS, ~10kb gzipped
  • No external dependencies

What is it?

WGLT is a JavaScript/TypeScript library for creating ASCII games in the browser.

WGLT is absurdly overoptimized for performance, using WebGL for minimal CPU.

WGLT is modelled after libtcod and rot.js.

Install

Install with npm (recommended)

Add dependency:

npm i -D wglt

Import library:

import * as wglt from 'wglt';

Link directly with HTML

Or, simply use the prebuilt:

<script src="https://unpkg.com/wglt@0.6.0/dist/cjs/index.cjs"></script>

Usage

Use it:

const term = new wglt.Terminal(document.querySelector('canvas'), 80, 45);
term.fillRect(0, 0, 80, 45, 0, wglt.Colors.YELLOW, wglt.Colors.DARK_BLUE);

let x = 10;
let y = 10;

term.update = function () {
  const moveKey = term.getMovementKey();
  if (moveKey) {
    x += moveKey.x;
    y += moveKey.y;
  }

  term.clear();
  term.drawString(1, 1, 'Hello world!');
  term.drawString(1, 3, 'Use arrow keys to move');
  term.drawString(x, y, '@');
};
